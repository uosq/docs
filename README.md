## What is MkDocs?
**MkDocs** is a fast, lightweight static site generator designed for project documentation. It uses Markdown files and a simple YAML configuration to produce clean, searchable documentation sites.

## Getting Started

### 1. Install MkDocs
pip install mkdocs

(Optional) Install the Material theme:
pip install mkdocs-material

### 2. Build & Serve the Docs
Run a local development server:
mkdocs serve

Build static HTML output:
mkdocs build

### Project Structure
docs/          # Markdown documentation files  
mkdocs.yml     # MkDocs configuration
